对比EXCEL，轻松学习PYTHON数据分析

张俊红

第1章 数据分析基础

1.数据分析指标分类

  • 总体概览指标：又称统计绝对数，反映某一数据指标的整体规模大小，关键性指标，决定公司的盈利情况。

  • 对比性指标：常见的就是同比、环比、差。同比是指相邻时间段内某一共同时间点上指标的对比，环比就是相邻时间段内指标的对比；差就是两个时间段内的指标直接做差，差的绝对值就是两个时间段内指标的变化量。

  • 集中趋势指标：通常用平均指标来表示。平均指标分为数值平均（普通平均/加权平均）和位置平均（中位数/众数）。

  • 离散程度指标：离散程度指标是用来表示总体分布的离散（波动）情况的指标，指标较大则数据波动大，反之则数据比较稳定。全距（又称极差）、方差、标准差用于衡量数值的离散情况。全距：用数据集中最大数（上界）减去数据集中最小数（下界）。方差是每个数值与均值距离的平方的平均值，方差越小说明各数值与均值之间的差距越小，数值越稳定。标准差是方差的开方，表示数值与均值距离的平均值。

  • 相关性指标：数据整体内的变量关系，一个变化时会引起另一个变化，反映这种关系的指标叫做相关系数，相关系数常用r来表示。
    $r(X,Y)=\frac{Cov(X,Y)}{\sqrt[2]{Var[X]Var[Y]}}$

    • 相关系数r的范围为[-1,1]；
    • r的绝对值越大，表示相关性越强；
    • r的正负代表相关性的方向，正代表正相关，负代表负相关。
  • 相关关系与因果关系：相关关系不等于因果关系，相关关系只能说明两件事情有关联，而因果关系是说明一件事情导致了另一件事情的发生。

2.数据分析常规流程

熟悉工具—-明确目的—-获取数据—-熟悉数据—-处理数据—-分析数据—-得出结论—-验证结论—-展示结论

第2章 Python基础知识

1.名词解释

  • IDE：Integrated Development Environment，集成开发环境。

2.基本概念

2.1 数

  • int:整数；
  • float:浮点数；

2.2 变量

变量：即变化的量，可以把它理解成一个容器，存储数据；变量区分大小写；

2.3 变量命名原则：

  • 变量名必须以字母或下画线开始，名字中间只能由字母、数字和下画线组成。
  • 变量名的长度不得超过255个字符。
  • 变量名在有效的范围内必须是唯一的。
  • 变量名不能是Python中的关键词。

2.4 Python的关键词
and elif import return
as else in try
assert except is while
break finally lambda with
class for not yield
continue from or
def global pass
del if raise

2.5 数据类型

  • type 查看数据类型

2.6 输出于输出格式设置

  • print
  • str.format
  • 一对一填充；一对多填充；
  • 浮点数设置： .2f
  • 百分数设置： .2%

2.7 缩进与注释

  • 单行注释#
  • 多行注释，多个#；’’’;”””;

2.8 字符串

  • 字符串可以用 + 直接连接；
  • 字符串的复制可以用 * N ，N为重复的次数；
  • 获取字符串的长度采用len函数；
  • 字符串的查找：in , not in , find;
  • 切片索引：[3]显示第4个；[:3]显示自己前面的，不包括自己，显示前三；[3:] 显示自己后面的，包括自己；[-3]倒数第三个；[:-3]显示自己前面的，不包括自己；[-3:] 显示自己后面的，包括自己；
  • 分割字符串split函数；
  • 移除字符strip函数;移除空格、换行符、指定字符；

2.9 数据结构——列表

  • 列表list：是用来存储一组有序数据元素的数据结构，元素之间用逗号分隔。数据元素在方括号中，列表是可变的数据类型，可添加、删除或者搜索列表中的元素。元素可以是int，str；
  • 列表的复制也可以用 * 操作符；
  • 列表的合并：用 + 操作符；extend函数，A.extend(B)讲B添加到A中；
  • 列表插入元素：append函数，在列表尾部插入元素；insert函数，在列表指定位置插入函数；
  • 列表中值出现的次数，使用count函数；
  • 获取列表中的位置，使用index函数；
  • 获取列表中指定位置的值：普通索引，切片索引；
  • 删除列表中的元素：pop函数，删除指定位置的元素；remove函数，删除指定元素；
  • 列表中元素排序，使用sort函数，默认为升序；降序为函数括号内填写reverse= True;

2.10 数据结构——字典

  • 键值对在字典中以{key1:value1,key2:value2}方式标记。键值对内部用冒号分隔，而各个对之间用逗号分隔；
  • 向字典输入键值用列表形式；
  • 字典可以将元组中的列表转换为字典键值；
  • keys()方法用来获取字典内的所有键；
  • values()方法用来获取字典内的所有值；
  • items()方法用来得到一组组的键值对，键对以列表形式体现，键值以元组形式体现；

2.11 数据结构-元组

  • 获取元组的长度，同列表用函数len();
  • 元组元素获取方法：普通索引，切片索引；
  • 元组转换为列表用list()函数；
  • 列表转换为元组用tuple()函数；
  • zip()函数用于将可迭代的对象（列表、元组）作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的列表；

2.12 运算符

  • 算数运算符：加 + 减 - 乘 * 除 / ，求余数 % ，乘方 ** ，商取整 // ；
  • 比较运算符：等于 == ，不等于 ！= ，不等于 <> ,大于 > ，小于 < ，大于等于 >= ，小于等于 <= ;
  • 逻辑运算符：与 and ，或 or ，非 not ；

2.13 循环语句

  • for循环：历遍；
  • while循环：重复；
  • if语句：条件满足作什么；
  • else语句：条件不满足作什么；
  • elif语句：else_if,多条条件判断；
  • 程序是自上而下运行；

2.14 函数

  • 函数是在一个程序中可以被重复使用的一段程序；
  • 函数的组成：函数名（必需）、参数、语句块（必需）、return、变量这几部分组成；
  • 函数定于语法如下：
    def 函数名（参数）：
      语句块
  • 函数参数：形式参数和实际参数；
